# Menubot

install requirements:

```
pip3 install -r requirements.txt
```

run:

```
export MENUBOT_WEBHOOOK_URL=https://staging.chatgrape.com/services/hook/custom/YYY/XXXXXXX/
python3 menubot.py --crawler gangl.GanglCrawler
```

or run all:

```
python3 menubot.py
```
